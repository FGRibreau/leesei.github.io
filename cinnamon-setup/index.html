Cinnamon setup

Settings

“Backgrounds”

“Advanced Options”
No picture, No gradient, use black

“Startup Application Preference”

Disable:

  • Caribou
  • Ctrl Alt Backspace
  • Desktop Sharing
  • mintUpdate
  • mintUpload
  • mintWelcome
  • Zeitgeist

“Keyboard” > “Keyboard Shortcuts”

Generally delete unused shortcuts.

“Windows”:

  • “Toggle Scale”: Ctrl+q
  • “Toggle Expo”: Ctrl+w
  • “Minimize window”: Super+KP_1
  • “Show Desktop”: Super+d
  • “Toggle maximization state”: Super+KP_9

“Window > Tiling and Snapping”:

  • “Push tile left”: Super+Left
  • “Push tile right”: Super+Right
  • “Push tile up”: Super+Up
  • “Push tile down”: Super+Down

“Window > Inter-workspace”:

  • “Move window to left workspace”: Shift+Super+Left, Super+comma
  • “Move window to right workspace”: Shift+Super+Right, Super+period

“Workspace > Direct Navigation”:

  • “Switch to workspace 1”: Super+1
  • “Switch to workspace 2”: Super+2

“System”:

  • “Lock screen”: Super+l

“Launchers”:

  • “Home folder”: Super+e
  • “Launch terminal”: Ctrl+Alt+t

“Applets”

Spices : Cinnamon (Applets)

~/.local/share/cinnamon/applets/

Force Quit II (white)
Force Quit (red)
Multi-Core System Monitor

Desktop Capture

Disable all key bindings
Don’t copy to clipboard
Enable all in notification, but no on click action

Sound With Apps Volume

Check “Volume 150%”
Uncheck “Mute with middle click”

Show Desktop ++

Set icon “desktop”, width 40
Opacify desktop icons

Timer With Notifications

Calender

Check “Show week numbers”
Custom date format: “%Y-%m-%d (%a) | %H:%M”

Menu

Check “Enable file system path in search box”

Notification

Uncheck “Show empty tray”

“Desklets”

Spices : Cinnamon (Desklets)

~/.local/share/cinnamon/desklets/

“Extensions”

Spices : Cinnamon (Extensions)

~/.local/share/cinnamon/extensions/

gTile

space
2x2, 3x3, 4x3, 4x4
check “Auto-Close”
uncheck “Animation”

Win7 Alt-Tab

“Themes”

Spices : Cinnamon (Themes)

~/.themes/

Download “Zukitwo”, use its border and control
Use “Numix” for desktop and icons
Uncheck “Show icons in menus”
Uncheck “Show icons on buttons”

Nemo

“Views”

Default: “List View”
List View zoom: “66%”

“Behavior”

Check “Ignore per-folder view preferences”

“Display”

Date format
Check “Show full path”
Check “Show advanced permissions”

“List Columns”

Check “Owner”, “Permissions”

“Preview”

Disable Text and Folder
Show thumbnails for < 10MB
Enable tooltips

“Toolbar”

Check “computer”, “terminal”
