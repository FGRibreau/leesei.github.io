Iterables/Generator

You can iterate over a list as many times as you want but you can iterate over a generator only once.
iterator: object with __next__() which returns a value, can call next() on it; it raises StopIteration exception when there are no more items
iterable: object with __iter__() which returns an iterator
generator: syntax sugar for iterator and iterable with yield expression
