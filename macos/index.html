Bootcamp

Boot Camp - Apple Support

If you have Windows install USB:

  1. Prepare another USB, formatted as FAT16
  2. Run "Boot Camp Assistant"
  3. Uncheck installation disk creation
  4. Reboot and install Windows
  5. Run Bootcamp.exe from the FAT16 USB

If you don't have Windows install USB:

  1. Prepare a 16G+ USB, download Windows ISO from Microsoft (Win7 Win10)
  2. Run "Boot Camp Assistant"
  3. Check installation disk creation
  4. Reboot and install Windows
  5. Run Bootcamp.exe under the USB's bootcamp/ folder
