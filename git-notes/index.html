Git notes

Git is a content addressable file system

  • workspace/working tree
    current file system
  • index/staging area
    add-ed, rm-ed, ready for commit
  • local repo
    committed, HEAD commit
  • upstream/remote
    a bare repo to push to
  • stash
    temporary area to store your changes
